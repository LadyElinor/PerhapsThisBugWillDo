# Same-Day Run Checklist — 72h Validation Launch

## Before noon (prep)
- [ ] Print-ready CLEAT-C1 and CLEAT-C2 sets completed and visually inspected
- [ ] Fasteners, spare hardware, and torque tool ready
- [ ] Incline board stable and inclinometer calibrated
- [ ] Ballast mass measured and labeled
- [ ] Current logger functional; time sync verified
- [ ] Phone storage/battery sufficient for all video captures

## Pre-run (at bench)
- [ ] Firmware/config hash recorded in data log
- [ ] Surface prep completed and depth leveled
- [ ] Slope set to 35° and verified
- [ ] Warm-up run completed (loaded)
- [ ] Randomized run order selected and written down

## During run block
- [ ] Start video + logger before every run
- [ ] Verbalize run ID at run start
- [ ] Record commanded/progress distances each run
- [ ] Record mean/peak current each run
- [ ] Flag any stall/overcurrent/damage event immediately
- [ ] Recondition surface between runs as needed

## Slope transition
- [ ] Wear photos taken for both candidates after 35° block
- [ ] Re-level surface and set slope to 45°
- [ ] Verify ballast and start posture unchanged

## End of day closeout
- [ ] All runs copied to `results/validation/2026-02-22/raw/`
- [ ] Master CSV template fully populated
- [ ] Quick analysis script executed and outputs saved
- [ ] Decision status recorded: CONTINUE / PIVOT / STOP
